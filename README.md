# Free the Tanker

A small tanker rescue word guessing game built with React, TypeScript, and Vite.

## View it on github pages 
https://farriskhalid.github.io/Free-The-Tanker/

## Run locally

```bash
npm install
npm run dev
```

## Build for a domain

```bash
npm run build
```

Upload the generated `dist/` folder to any static host, or connect this folder to a service like Netlify, Vercel, GitHub Pages, or Cloudflare Pages.
